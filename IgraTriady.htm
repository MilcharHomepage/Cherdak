Триады

Для игры требуется доска 12х12 и примерно по 60-70 белых и чёрных шашек. Проще всего использовать комплект для игры го.

Перед началом игры шашки расставляются следущим образом:



Начинают игру белые. Каждым ходом игрок выставляет на доску две шашки так, чтобы в результате образовался хотя бы один ряд из трёх шашек (триада) по вертикали или горизонтали. Строить ряды из четырёх и более шашек запрещено. Диагональные ряды не считаются. Цель игры — выстроить как можно больше триад.

Если кто-то из игроков не может своим очередным ходом построить триаду, он пропускает ходы до конца игры. Как только другой игрок также не может сделать очередной ход — игра заканчивается. Подсчитывается количество триад у каждого игрока. У кого их больше — тот победил.
